Requis pour fonctionner : JRE 7 minimum

---- Instructions Windows ----

Faites "MAJ+Clic droit" dans le dossier courant, puis faites "Ouvrir une fenêtre de commandes ici".
Tapez ensuite l'instruction "dungeontrouble nbJoueurs numNiveau", avec nbJoueurs le nombre de joueurs
(soit 1 soit 4) et numNiveau le numéro du niveau (de 1 à 3). Enjoy !

---- Instructions GNU/Linux ----

Ouvrez un terminal dans le dossier courant, puis faites soit "./dungeontrouble.sh nbJoueurs numNiveau", 
soit "bash dungeontrouble.sh nbJoueurs numNiveau", avec nbJoueurs le nombre de joueurs
(soit 1 soit 4) et numNiveau le numéro du niveau (de 1 à 3). Enjoy !
